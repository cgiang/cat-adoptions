# Cat Adoptions Analysis

Analyzing factors that influence cat adoption likelihood and length of stay (LOS) using shelter data.

## Project goals

- Identify drivers of faster adoption
- Analyze length of stay by age, color, and intake condition
- Simulate A/B tests for adoption descriptions
- Build a predictive model for adoption within 30 days

## Structure

- `notebooks/` — exploration and analysis
- `src/` — reusable ETL, features, modeling logic
- `sql/` — analytical queries
- `dashboards/` — interactive visualization
- `data/` — raw and processed data (not committed)
